Como gerar a tua chave Worten

2 minutos · 3 passos. Vais precisar de uma conta Worten Seller Center com permissões de administrador.

1

Inicia sessão no Worten Seller Center

URL: sellercenter.worten.pt

Abre o Seller Center num separador novo e inicia sessão com as credenciais habituais. Se ainda não tens conta com permissões para gerar chaves API, contacta o admin da tua conta Worten antes de continuar.

step-1.png · ecrã de login do Seller Center
2

Abre as definições de API

Caminho: menu da conta (canto superior direito) → Definições → API

No menu da conta, abre Definições e depois API. Vais ver a lista de chaves activas na tua conta — provavelmente vazia se ainda não geraste nenhuma.

step-2.png · navegação até à secção API
3

Gera a chave e copia-a

Botão: Gerar nova chave API

Clica em Gerar nova chave API. Dá-lhe um nome — recomendamos MarketPilot para saberes facilmente para que serve. A chave vai aparecer uma única vez: copia-a imediatamente e cola-a no MarketPilot. Se a perderes, terás de gerar uma nova.

step-3.png · ecrã com chave gerada e botão copiar
lock

A tua chave dá acesso total à conta Worten — incluindo dados bancários, encomendas e preços. No MarketPilot, fica encriptada em repouso. Apenas o motor de repricing a usa para falar com o Worten — nem o nosso fundador a vê em texto puro.
